# Aditaya Kumar Mishra Portfolio

Premium React portfolio powered by Vite.

## Scripts

Run the local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deployment

Vite builds the production site into `dist/`. Public assets and SEO files are kept in `public/` and copied into the production output.
